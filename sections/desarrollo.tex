\section{Dessarrollo de la Practica}

\subsection{Comandos usados}
Los comandos se tomaron principalmente de \cite{comandos_cisco} y se usaron especificamente
\begin{minted}{bash}
# Para mostrar el sistema operativo, version y direccion fisica
Router> show version 
# Para mostrar que interfaces
Router> show ip interfaces brief 
# Para mostrar las direcciones de las interfaces
Router> show interfaces 
\end{minted}
Adicionalmente, recomendado por el docente encargado, si algo de lo anterior falla en mostrar la 
informacion.
\begin{minted}{bash}
Router> show tech-support 
\end{minted}
Este ultimo comando se uso especialmente las direcciones fisicas.

Tambien, dado que uno de los dispositivos no era Cisco, se uso la guia de usuario
\cite{3com4500_config_manual} para encontrar los comandos requeridos:
\begin{minted}{bash}
# Para mostrar el sistema operativo, version y direcciones fisica
Router> display version
Router> display ip interface
\end{minted}
